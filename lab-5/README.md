## Опис завдання

Розробка інтерактивного веб-додатку (дашборду) за допомогою бібліотеки **Streamlit** для зручної візуалізації та аналізу даних вегетаційних індексів (VCI, TCI, VHI) з Лабораторної роботи №2. Додаток дозволяє зовнішнім користувачам фільтрувати дані за часом та регіонами, а також порівнювати показники різних областей за допомогою графіків та таблиць.

## Структура та функціонал додатку

Інтерфейс розділено на дві колонки:
1. **Бічна колонка (Керування):** Містить усі інтерактивні елементи керування (віджети).
2. **Основна колонка (Відображення):** Містить три вкладки (Tabs) для перегляду результатів (Таблиця, Графік регіону, Графік порівняння).

### Доступні віджети керування:
* **Dropdown (Індекс):** Вибір часового ряду для аналізу (VCI, TCI або VHI).
* **Dropdown (Область):** Вибір конкретної області України.
* **Slider (Тижні):** Вибір інтервалу тижнів.
* **Slider (Роки):** Вибір інтервалу років.
* **Checkbox (Сортування):** Два прапорці для сортування даних у таблиці за зростанням або спаданням обраного індексу. *(Примітка: реалізовано захист від одночасного вибору обох типів сортування)*.
* **Button "Reset":** Кнопка для миттєвого скидання всіх фільтрів до їхніх початкових значень.
