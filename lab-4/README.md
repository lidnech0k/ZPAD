## Опис завдання

Створення програми з інтерактивним інтерфейсом для малювання графіка функції гармоніки (функція виду `y(t) = A*sin(ωt + φ)`) з накладеним шумом. Реалізовано можливість змінювати параметри гармоніки та шуму за допомогою слайдерів, кнопок та чекбоксів. Зашумлену гармоніку відфільтровано за допомогою цифрового фільтру з бібліотеки `scipy.signal`. Проведено візуальне порівняння "чистої", зашумленої та відфільтрованої гармонік за допомогою бібліотеки `matplotlib`.

## Вимоги до системи

* **ОС:** Windows, macOS або Linux
* **Мова:** Python 3.10+

## Встановлення залежностей

* `numpy`
* `matplotlib`
* `scipy`
  
## Інструкція користувача

* **Слайдери гармоніки:** Дозволяють змінювати амплітуду (`amplitude`), частоту (`frequency`) та фазовий зсув (`phase`). 
* **Слайдери шуму:** Керують середнім значенням (`noise_mean`) та дисперсією шуму (`noise_covariance`). При зміні параметрів шуму генерується новий шум, а при зміні параметрів гармоніки — існуючий шум просто накладається на новий сигнал.
* **Чекбокс "Show Noise":** Вмикає або вимикає відображення шуму на графіку (якщо вимкнено — відображається "чиста" гармоніка).
* **Елементи фільтрації:** Дозволяють увімкнути відображення відфільтрованого сигналу поруч із початковим для порівняння.
* **Кнопка "Reset":** Відновлює всі параметри гармоніки, шуму та стан чекбоксів до початкових значень.
